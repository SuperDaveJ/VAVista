2.2 Routines (Routine Structure and Format)

The Routine is the basic unit of M code and contains lines of M code and comments related to a common functionality.

EXAMPLE1      ;FO-OAK/JLI - EXAMPLE ROUTINE ;09/13/2011  08:45
            ;;1.0;PACKAGE;**1,2**;SEP 13, 2011;BUILD 1
            ;Per VHA Directive 10-92-142, this routine should not be modified
            Q
            ;
ENTRY1         ;
            W !,"ENTERED AT ENTRY1"
            Q
            ;
ENTRY2(VALUE)        ;
            W !,"ENTERED AT ENTRY2 WITH VALUE=",VALUE
            Q
