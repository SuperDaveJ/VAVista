2.2.6 Restricted Routine Names

The names of routines in a package start with that package’s namespace (e.g., XU for Kernel). The following package routine names are restricted (where * indicates following letters or digits):

  • NAMESPACE_I* (with the exceptions of Kernel, VA FileMan, and routines created to support the INIT process). Thus, in namespace MAN, MANI4 could not be used as a routine name.
  • NAMESPACE_NTE* (with the exception of the package integrity routines). Thus, in namespace MAN, MANNTEX could not be used as a routine name.
  • NAMESPACE_Z*. Thus in namespace MAN, MANZX could not be used as a routine name.
