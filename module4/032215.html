2.2.10 Naked References (cont.)

  • An appropriate preceding full reference is one that is on the same physical routine line as the naked reference and has no code between it and the naked reference that branches in any manner to other lines of code or executables.
  • Naked references requiring documentation must be documented within the routine in the immediate vicinity of the naked reference. Naked references that are preceded by a full reference that is outside of the routine where the naked reference is used must have documentation in both the routine containing the full reference and the routine containing the naked reference. This documentation must be in the immediate vicinity of the appropriate reference.
  • Uses of naked references in called utilities are exempt, e.g.,
    S DIC=200,DIC(0)="AEQ",DIC("S")="I $L($P($G(^(1)),"^",9))" D ^DIC
    is a legitimate use of the naked reference.
