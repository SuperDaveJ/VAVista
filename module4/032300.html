2.3  Variables

  • A variable is a symbolic name that is used to reference a data value. It can be considered a symbolic address that is associated with a data value.
  • Variable names start with a single alphabetic character optionally followed by additional characters.
  • Example:  DOB, which would contain a date for Date of Birth
